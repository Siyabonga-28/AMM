# AMM – All Maintenance Matters

## Overview

AMM (All Maintenance Matters) is a responsive business website built for a property maintenance company.  

This project demonstrates frontend development skills, interactive UI design, state management using the Local Storage API, and admin dashboard functionality.

The website simulates a real-world service booking system with an admin panel for managing bookings.

---

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6+)
- Local Storage API
- Responsive Design Principles

---

## Features

### User Side
- Modern landing page with hero section
- Services overview
- Pricing packages
- Interactive booking calendar
- Time slot selection system
- Contact form with validation
- File upload UI (image preview)
- Data persistence using Local Storage

### Admin Dashboard
- View submitted bookings
- Manage booking data
- Update or delete entries
- Simulated data management using Local Storage

---

## Booking System Logic

- Users select a date and available time slot.
- Booking details are stored in the browser using Local Storage.
- The admin dashboard retrieves and displays stored bookings.
- Data updates dynamically without page reload.
